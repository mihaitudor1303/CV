# Online CV Website

This repository contains the source code for my online CV website. It includes HTML, CSS, and JavaScript files to create a responsive and interactive personal CV.

## Project Structure

- `index.html` - The main HTML file for the website.
- `styles/` - Directory containing CSS files for styling the website.
    - `main.css` - Main stylesheet for the website.
- `scripts/` - Directory containing JavaScript files for adding interactivity.
    - `main.js` - Main JavaScript file for the website.

## How to Use

1. Clone the repository:
     ```
     git clone https://github.com/yourusername/your-repo-name.git
     ```

2. Open `index.html` in your web browser to view the CV website.

## Features

- Responsive design that works on both desktop and mobile devices.
- Interactive elements powered by JavaScript.
- Clean and modern design using CSS.
